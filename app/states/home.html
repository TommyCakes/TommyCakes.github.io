Feed Me!

I had the challenge of using a unknown API to pull back a database full of recipes. Having learnt Angular I incorporated the use of the built in router to create a single page app with almost no loading time. The user supplies a query as a particular food item and then are presented with a list of recipes containing there item which they are able to interact with to get more information about the recipe.
